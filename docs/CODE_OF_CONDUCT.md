# Code of Conduct

## 1. Required Behavior
Interaction in this repository must focus strictly on:
- Physical modeling accuracy.
- Mathematical invariants and numerical stability.
- Direct code functionality.

## 2. Prohibited Behavior
The following actions violate repository rules:
- Personal insults or derogatory statements.
- Harassment or exclusionary conduct.
- Publishing private information (Doxing).

## 3. Enforcement Mechanism
Violations of these rules must be documented mathematically or procedurally. 
If an individual violates conduct rules, report it by opening a GitHub Issue labeled `conduct-violation`.

Repository maintainers are the sole arbiters of resolution. Any action taken will be executed through GitHub Issue closures or repository access blocking.
